Tên đề tài :Web Cửa Hàng Mỹ Phẩm  
    Giới thiệu :
        Dự án xây dựng một website bán mỹ phẩm trực tuyến nhằm đáp ứng nhu cầu mua sắm tiện lợi và an toàn của người tiêu dùng. Website được thiết kế với giao diện thân thiện, cho phép người dùng dễ dàng xem sản phẩm, tìm kiếm, đặt hàng và quản lý giỏ hàng. Đồng thời, hệ thống cung cấp trang quản trị giúp người bán dễ dàng quản lý sản phẩm, danh mục, đơn hàng, thông tin khách hàng và doanh thu một cách khoa học.

    Công nghệ sử dụng 
        Nền tảng mã nguồn mở/CMS: WordPress 
        Plugin Thương mại điện tử: WooCommerce 
        Ngôn ngữ lập trình & Giao diện: PHP, HTML, CSS, JavaScript
        Hệ quản trị cơ sở dữ liệu: MySQL 
        Môi trường phát triển cục bộ: XAMPP (Localhost) 
        Máy chủ triển khai (VPS): GenCloud (sử dụng ổ cứng NVMe) 
        Các Plugin hỗ trợ khác: VietQR,...
    Hướng dẫn cài đặt
        Tải và cài đặt phần mềm XAMPP 
        Tải mã nguồn website đã nén của nhóm.  
        Trong thư mục htdocs của XAMPP (đường dẫn mặc định thường là C:\xampp\htdocs) tạo 1 folder là "myweb" copy toàn bộ thư mục chứa mã nguồn website vào trong 
        Mở bảng điều khiển XAMPP  và nhấn nút Start để khởi động 2 dịch vụ là Apache và MySQL.  
        Truy cập vào địa chỉ http://localhost/phpmyadmin trên trình duyệt và tạo một cơ sở dữ liệu (Database) mới. 
    Hướng dẫn chạy project
        Import file cơ sở dữ liệu (đuôi .sql) của hệ thống vào database bạn vừa tạo ở bước trên.
        Mở file wp-config.php nằm trong thư mục gốc của dự án, tiến hành khai báo lại các thông tin kết nối database (bao gồm Tên DB, Username, Password).  Lưu lại file cấu hình. 
        Mở trình duyệt và truy cập vào đường dẫn: http://localhost/myweb để vào trang chủ website. 
        Để vào trang quản trị hệ thống, truy cập theo đường dẫn: http://localhost/myweb/wp-admin.
